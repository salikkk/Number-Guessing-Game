# Number-Guessing-Game
This is my second C project. a Number Guessing Game where the player tries to guess a random number generated by the computer.

The computer generates a random number between **1 to 100**, and the user has to guess it.  
After each guess, the program tells whether your guess is **Too High** or **Too Low**.
